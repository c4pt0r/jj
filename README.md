##Overview

Redis alike NoSQL Memory DB for JSON document, support JSONPath

``
JDOCSET key jsondocstring
JDOCGET key

JSET key path value
JGET key path

JPUSH key path value
JPOP key path

JINCR key path num

JREM key path
DEL key

JKEYS keyprefix[*]

BGSAVE

LOAD

SAVE
``
