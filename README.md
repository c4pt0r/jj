# jj

##Overview

Redis alike NoSQL Memory DB for JSON document, support JSONPath

```
jdocset [key] [val]
jdocget [key]

jset [key] [json path] [val]
jget [key] [json path]

jincr [key] [json path] [integer]

jpush [key] [json path] [val]
jpop [key] [json path]

save
bgsave
```

* Python APIs
* Golang APIs
